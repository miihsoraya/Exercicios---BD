## 📌 Sobre

Este repositório reúne conteúdos acadêmicos relacionados à área de **Banco de Dados**, com foco na prática e consolidação de conhecimentos.

Serão disponibilizados materiais como:

- 💾 Scripts SQL  
- 📊 Resultados de consultas  
- 🧪 Testes e experimentações  
- ⚙️ Configurações de ambiente com Docker  

---
## 📌 Ambiente Docker

Inicie os containers via Docker

Acesse o pgAdmin:
  - http://localhost:8080

Navegue até:
  - Servers → Databases → postgres

Abra o Query Tool.
Execute os scripts necessários.
